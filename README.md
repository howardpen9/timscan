# Ton Blockchain Explorer
- MainNet: https://tonlens.com/
- TestNet: http://Testnet.tonlens.com

## Installation

```
git clone https://github.com/catchain/tonscan.git
cd tonscan
npm install
```

## Building

```
cp .env.example .env
npm run prod
```

### Testnet version

```
cp .env.example .env.testnet
npm run prod_testnet
```
